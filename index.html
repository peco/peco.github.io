What is peco

peco is a simple tool that allows you to interactively filter your data.

Get peco

If your on OS X and are using homebrew, use brew tap to get the repository

$ brew tap peco/peco
$ brew install peco

If you have already tapped the repository and want to upgrade, use brew update.

    Note: If you have previously tapped my personal repo lestrrat/peco,
    you will have to untap and tap the peco/peco repository
